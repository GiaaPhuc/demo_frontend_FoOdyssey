# 🥗 demo-frontend-FoOdyssey

**FoOdyssey** là giao diện ứng dụng **du lịch ẩm thực thông minh**, được xây dựng nhằm **trình diễn khả năng ứng dụng AI trong việc khám phá ẩm thực địa phương**.

Repository này **tập trung hoàn toàn vào Frontend**, nhấn mạnh vào:

- Trải nghiệm người dùng (UX)
- Thiết kế giao diện hiện đại (UI)
- Hiệu ứng mô phỏng AI tiên tiến
- Kiến trúc Frontend chuyên nghiệp

# ⚙️ Hướng dẫn cài đặt nhanh

## 1. Clone Repository

```bash
git clone https://github.com/phucvo-vn/demo-frontend-FoOdyssey.git
cd demo-frontend-FoOdyssey
```

---

## 2. Cài đặt Dependencies

```bash
npm install lucide-react framer-motion clsx tailwind-merge
npm install -D tailwindcss@latest @tailwindcss/postcss@latest postcss autoprefixer
```

---

## 3. Chạy Development Server

```bash
npm run dev
```

Ứng dụng sẽ chạy tại:

```
http://localhost:3000
```
